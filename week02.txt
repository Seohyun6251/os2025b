리눅스 명령어
- ls
- ls 옵션a
- touch / 새로운 파일 만들기
- cd 폴더명
- cd ..
- cat 파일명

깃 명령어
- git init / 현재 폴더를 git이 관리하는 폴더로 만듦
- git status / 현재 git이 관리하는 파일 상태확인
- git add 파일명 / 스테이지에 올리는 명령어
- git add .
- git config --global user.email "개인이메일주소"
- git config --global user.name "ID"
- git config --list
- git commit -m "커밋에 남길 메세지"
- git log
- git checkout 해시코드전체 또는 7자리 / 해당 시점으로(커밋으로)이동
- git checkout - / 이동했다가 다시 돌아옴
